# Cage Volume Calculator

`cage_calc.py` calculate the volume of the cage cavity and saves this information to a cage information file.

Uses [CageCavityCalc](https://github.com/VicenteMartiCentelles/CageCavityCalc).

## Usage

```
python cage_calc.py <cage_pdb_file> <cage_info_file>
```

## Cage Information File Structure

A cage information file will follow the format shown in this example:

```
name: zd001; Picture 1
cavity volume (A3): 329.453125
```
